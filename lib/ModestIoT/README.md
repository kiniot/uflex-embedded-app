# ModestIoT

ModestIoT is a lightweight, object-oriented, event-driven, and CQRS-inspired nano-framework for Arduino-compatible devices such as the ESP32.

This README is a repository-local integration note. It is not presented as official upstream documentation for the original framework.

In this repository, the framework is kept as a separate module under `lib/ModestIoT/` so it can remain clearly distinguishable from the rest of the uFlex application code. Its role is to provide the conceptual foundation for devices, sensors, actuators, commands, and events.

The current contents of this directory are being preserved in a form intended to stay close to the original framework layout and naming conventions.
